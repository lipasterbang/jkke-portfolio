Sistem Carian Jadual Kadar Kerja Elektrik (JKKE 2024)

Portfolio Projek Digital

Objektif Sistem

Sistem ini direka untuk memudahkan proses carian kadar kerja elektrik dan penjanaan sebut harga mengikut format rasmi Jabatan Kerja Raya (JKR).

Ciri-ciri Utama:

  • Carian Kod JKKE berdasarkan JSON Data
  • Penjanaan Bill of Quantities (BQ) dalam format rasmi
  • Output PDF & Excel automatik
  • Kiraan caj tambahan berdasarkan jarak projek
  • Pengurusan data melalui localStorage / database (akan datang)

Teknologi Digunakan:

HTML, CSS, JavaScript, JSON, PDFMake, SheetJS (xlsx.js)
