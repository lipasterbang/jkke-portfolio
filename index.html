JKKE 2024 - Sistem Carian Jadual Kadar Kerja Elektrik

📌 Tentang Sistem

Sistem Carian JKKE 2024 membantu kakitangan JKR membuat carian kadar kerja elektrik mengikut kod yang telah ditetapkan, serta menjana sebutharga secara automatik berdasarkan format rasmi.

🚀 Ciri-ciri Utama

  • Carian kadar kerja berdasarkan Bab dan Subtopik
  • Penjanaan sebutharga automatik (Auto Quotation)
  • Papar dan eksport format Jadual Kadar Kerja seperti dokumen rasmi
  • Integrasi sistem dengan fail JSON (akan datang: MySQL)

🔗 Pautan Sistem

Halaman Utama (jkke.html) Penjana Sebutharga Lihat Semua Kod JKKE

🛠️ Teknologi & Perancangan

  • Frontend: HTML, CSS, JavaScript
  • Data: JSON (akan dipindah ke MySQL untuk sistem lebih efisien)
  • Hosting: GitHub Pages + QR Code untuk akses mudah
